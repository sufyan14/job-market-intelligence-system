# Raw Data Setup

The raw datasets are not included in this repository due to size constraints.

## Required Files

Place the following files in this directory:

- linkedin_jobs.csv
- indeed_jobs.csv

## Data Sources

- LinkedIn Job Postings (2023 - 2024): Kaggle
- Indeed Job Postings: Kaggle

## Notes

- Filenames must match exactly.
- CSV encoding should be UTF-8.
